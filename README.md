# SDLSHOOTER

- 一个基于SDL2的简单射击游戏
- 一个个人学习项目
- 根据bilibili教程（BV号：BV1wSCFYQEyc）完成

## 运行

- 安装SDL2并且将环境变量配置好
- 使用CMake编译
- 运行可执行文件

## 游戏控制

- 基础移动：WASD控制方向
- 射击：空格射击
- 返回开始界面：ESC
- 全屏切换：F4
- 开始界面：按下空格开始
- 结束界面
	- 输入名字（不知道为什么有时候输入法调不出来），支持中英文
	- 回车确认名字
	- 空格再来一把
